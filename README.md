# REMEMBER TO USE THE BRANCH SYSTEM

### Go to a branch
```git
git checkout -b "25-w-<WEEK>-<VER>"
```

### Add files to be commited
```
git add <FILES> // or optionally, just a . for all the updated files
```

### Commit them.
```
git commit -m "<MESSAGE>"
```
